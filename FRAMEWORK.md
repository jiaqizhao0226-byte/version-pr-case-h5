# 游戏版本舆情洞察与公关复盘框架 (PR Analysis Framework)

本项目建立了一套用于拆解和复盘游戏版本舆情危机的标准化分析框架。通过对行业内具有代表性的重度舆情事件进行解构，提炼出可复用的公关应对经验与教训。

## 一、 舆情分级矩阵 (声量 vs 伤害)

评估一场舆情的性质，不能仅看“骂得有多凶（声量）”，更要看“是否动摇了付费与留存根基（伤害）”。我们将其划分为三大类：

1. **🔥 核心危机（高伤害 + 高声量）**
   - **特征**：全网炎上，且玩家付诸于实质性的退款、停氪、融号或消保投诉。通常涉及“核心资产剥夺”或“商业欺诈”。
   - **公关应对**：必须在黄金24小时内光速滑跪，且必须伴随超越常规的天价补偿（血包）来切断负面循环。绝不能采用“官方套话”或冷处理。

2. **🌪️ 舆论风暴（低伤害 + 高声量）**
   - **特征**：频上热搜，全网道德审判（如涉嫌擦边、软色情）。但参与抵制的往往是外围路人或云玩家，游戏的核心付费圈层反而可能会出于逆反心理坚决护盘。
   - **公关应对**：保持克制，采取冷处理或拖延战术。**绝对不要为了迎合不花钱的“路人/键盘侠”去删改核心玩家花钱想看的内容**，否则会引发真正的核心危机。

3. **🧊 隐性流失（高伤害 + 低声量）**
   - **特征**：社区讨论度一般，没有上热搜，但玩家感到严重的安全感缺失（如长线运营的资产清零焦虑），导致默默退坑。
   - **公关应对**：需要敏锐的数据洞察，官方应主动出面，通过极其详尽的长文或视频解释底层机制，给玩家派发“定心丸”。

## 二、 玩家声音定性分析

舆情爆发时，官方需要甄别声音的性质，以决定是“顺从”还是“切割”：
*   **真实痛点**：玩家的核心体验（数值、手感、金钱）切实受损。必须正面解决，不能用补偿来搪塞核心问题的修复。
*   **圈层噪音 / 道德审判**：特定群体基于自身立场（如性别对立、女拳/媚男指控）发起的攻击。这类声音极大，但极易造成“粉红红利反噬”。官方需明确自身的核心受众画像，切忌“既要又要”。

## 三、 核心矛盾标签体系 (Tag System)

我们通过对过往案例的沉淀，提炼出以下高频致命的舆情标签：

*   **[核心资产剥夺]**：变相削弱已发售角色、实装与PV不符、数值造假。这是游戏运营的绝对红线。
*   **[情感背叛 / OOC]**：区域区别对待、人设崩塌、NTR感（如女武神给外服跳兔女郎舞）。二游玩家极其看重角色的“纯洁性”与“排他性”。
*   **[底层契约撕毁]**：突击涨价、战令收费变现金、外观强行绑定角色。破坏了玩家默认的商业安全感。
*   **[教玩家玩游戏]**：通过全局无差别削弱、增加操作门槛等方式强行拉长玩家在线时间，剥夺玩家的“割草爽感”和权力幻想。
*   **[价值观冲突]**：文案雷点（如抢人头嘲讽“你怎么还在”、美化逃跑将军）、性别对立。

## 五、 PR 应对操作指南 (事后诸葛亮)

1.  **公告透明度是免死金牌**：任何涉及玩家习惯、体验、数值的改动，必须白纸黑字写在更新说明里。“说出来是被骂，不说出来被发现就是欺诈”。
2.  **用“工业事故”阻断“主观作恶”**：当玩家怀疑官方在“恶意暗改逼氪”时，最好的自救方法是承认“我们代码管理太菜出了事故”。玩家愿意原谅草台班子，但绝不原谅态度的作恶。
3.  **大节点决不能精算福利**：周年庆、春节等关键时间点，宁可平时抠，大节点也必须大方。预期的落差会引发毁灭性的全球差评轰炸。
4.  **真诚的视频比长文更有穿透力**：在触及底盘的信任危机面前，主策划以诚恳认错的姿态发布视频，能最大程度消耗掉玩家的戾气，把“对抗”拉回“对话”的维度。
