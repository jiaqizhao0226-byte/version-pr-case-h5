# 游戏版本舆情洞察与案例库 (Game PR Case H5)

这是一个用于展示和分析游戏行业重大舆情危机的 H5 看板项目。通过结构化的数据和清晰的前端界面，深度复盘官方公关的“得与失”。

## 📚 核心分析框架与理论沉淀
我们在梳理了大量案例后，沉淀出了一套行之有效的**“舆情分级与公关复盘框架”**，涵盖了“声量与伤害矩阵”、“玩家声音定性”、“核心矛盾标签体系”等内容。

👉 **[点击查看完整的《游戏版本舆情洞察与公关复盘框架》文档](./FRAMEWORK.md)**

## 🚀 项目概览
- **多维度筛选**：支持按照舆情严重程度、市场、问题性质（如：核心资产剥夺、教玩家玩游戏）进行交叉筛选。
- **深度复盘页面**：
  - **T-Window 时间线**：还原官方与玩家的攻防拉扯（左右交错时间线）。
  - **玩家痛点剖析**：精准定位核心矛盾与玩家愤怒的深层原因。
  - **PR 应对与启发**：拆解官方具体动作的 Pros & Cons，输出可迁移的公关建议。

## 🛠️ 技术栈
- 纯原生 HTML / CSS / JavaScript 构建，无依赖，即插即用。
- 数据存储在 `data/cases.json` 与 `cases/*.json` 中，方便快速扩充。
